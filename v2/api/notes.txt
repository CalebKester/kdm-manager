Responsive webapp with a few breakpoints
Some kind of js framework for client-side fanciness


### API Stack
 - systemd (service and socket)
 - gunicorn (using a dummy wsgi entry point)
 - flask (for routes)


### Routes
 - update user
 - update survivor
 - update settlement


### Dependencies

apt install python-pip gunicorn ubuntu-standard systemd-sysv 
pip install Flask
